DigiDive is a frontend web project for a minimal, user friendly e-stationery store. It focuses on clean design, warm color tones, and a calm shopping experience.
## Tech Stack
- HTML5
- CSS3

## Features
- Landing page with background image and tagline  
- Shop by category (Planners, Journals, Stickers)  
- Minimal layout with aesthetic color palette  
- Footer with navigation and payment icons  

## Run Locally
Open `index.html` in your browser.
